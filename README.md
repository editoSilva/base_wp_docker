## - *Base Docker - Wordpress*

## - Comando Utilizado

*Copie e cole no terminal*

```sh
cd docker-compose up -d --build

```
## - *URL de Acessos*

- Site:

```sh
http://localhost:8000

```

- PhpMyAdmin:

```sh
http://localhost:8080

```

- Login: 

```sh
  wordpress
```

- Senha: 

```sh
  wordpress
```
